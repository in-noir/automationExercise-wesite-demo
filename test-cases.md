

 TC_01 - Verify user can successfully sign up

Steps:
1. Open website- https://automationexercise.com/
2. Click Signup/Login
3. Enter valid name and email
4. Click Signup button
5. Fill all required details
6. Click Create Account

Expected Result:
- Account should be created successfully
- User should be redirected to home page
- "Logout" option should be visible


## TC_02 - Verify error when signing up with existing email

Steps:
1. Open website- https://automationexercise.com/
2. Click Signup/Login
3. Enter name and already registered email
4. Click Signup

Expected Result:
Error message "Email Address already exist!" should be displayed
